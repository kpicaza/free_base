Inspired by Bootstrap contrib theme from https://drupal.org/project/bootstrap

Instalation:

Dependencies jquery_update module https://drupal.org/project/jquery_update

drush dl jquery_update
drush en jquery_update

Copy freeee_subtheme folder to sites/all/themes/THEME_NAME
Rename freeee_subtheme.info.starterkit on THEMENAME.info and change name by your theme name.
Download pre-compiled bootstrap and unpack it in your new theme folder as bootstrap.
